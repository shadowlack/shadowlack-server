We try our best to create an enjoyable experience for all players.

We approach things from an accessibility-first standpoint. Given that we're a text-based game, we've has always had a higher amount of sight-impaired players, as well as those with motor impairments (RSI, cerebral palsy, muscular dystropy, etc).

Key Accessibility Features

Shadowlack is built with consideration for screen readers and a wide range of assistive technology.

  • Keyboard Navigation: You can navigate Shadowlack entirely with a keyboard.
  • Touch-Screen Friendly: Everything will work on your tablet or mobile device.
  • Dark and Light Mode: Both light and dark site themes are available to switch between. A theme switched can be found in the footer of the site.
  • Dyslexia and Reading Disorders: Shadowlack attempts to use your device's native font stack wherever possible.
  • Screen Readers: It is our hope that Shadowlack remains highly accessible for all screen reader technology, whether they be on a computer, tablet, or mobile device.
  • Best Practices: We try to follow WAI-ARIA Authoring Practices wherever it is possible.
  • English as a Second Language: Whether English is your second, third, or 12th language – you are welcome here. Learn, practise and improve upon your English.

Client Commands

Screen reader mode is set at the session level, so it will need to be set every time that you connect. Though we hope to make this setting persistent.

Command What it does
option screenreader = onTurns screen reader mode on. This will strip ANSI and superflous text graphics from the client.
option screenreader = offTurns screen reader mode off.
